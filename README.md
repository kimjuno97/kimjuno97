## 반갑습니다, 저는 문제를 즐기는 개발자 김준호입니다 👋

> "모르는 영역이 나와도, AI와 함께라면 겁나지 않습니다."

낯선 문제도 주저 없이 마주하고,
단순 반복 작업은 AI에 맡겨 본질적인 문제에 집중하는 개발자입니다.

Node.js + TypeScript 기반 서버 개발 경험이 있으며,
Flutter/React 프론트와 함께 풀스택으로 다뤄왔습니다.
PortOne 결제 연동, Socket.io 기반 실시간 채팅,
레퍼런스가 없는 레거시 기반 본인인증 모듈 WebView 연동 경험이 있어요.

바이브코딩(AI 협업 개발)을 활용해 빠르게 프로토타입을 만들고,
실제 사용자 피드백을 받으며 개선한 경험이 있습니다.

## 🛠 Tech Stack

**프론트엔드**
- Flutter (Bloc, Riverpod)
- React / Next.js

**백엔드**

- GraphQL / REST API

- Firebase (Auth / Firestore / Storage / Functions)
